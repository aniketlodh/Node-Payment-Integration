Project : Payment Gateway Integration <br>
  
A web app with payment gateway integrated into it for donating money to unpriviledged children.<br>

Tech Used:<br>

Frontend:- HTML, CSS, Bootstrap, EJS <br>

Backend:- NodeJs, Express <br>

3rd Pary Gateway used:- Instamojo. <br>

Once the payment is successful, a confirmation email with the invoice of successful payment will be sent to the payer's email id.
